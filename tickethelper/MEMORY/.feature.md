「搶票助手 Pro」是 Chrome Extension，同時支援 KKTIX、Tixcraft、inline.app 三個訂位網站。設計理念：一個側邊面板 popups.html, popup.js;KKTIX/tixcraft/inline-content.js/content scripts 各一;background.js service worker 負責恢復頁面重載。

核心機制:
- globalEnabled toggle(全域開關)控制所有內容腳本注入，popup.js共用 toast/showToast/kktixAddLog/tcAddLog/inAddLog等日誌函式族。
- background.js在changeInfo.status==="loading"/"complete"時偵測 URL 模式(KKTIX_PATTERN/TIXCRAFT_PATTERN/INLINE_PATTERN),並依據平台重載策略 (KKTIX: reload→注入+START;Tixcraft: loading→立即注入 +START;Inline: complete→注入)自動恢復流程。
- tixcraft-content.js透過 chrome.runtime.sendMessage{action:"OCR_REQUEST"}呼叫 background 代理 HTTP 到本機 ocr_server.py,繞過 HTTPS->HTTP fetch mixed content restriction。
- Discord webhook notification在到達 checkout/verify頁時由 MutationObserver偵測並送出通知 (KKTIX/Tixcraft各自有 webhook URL)。
